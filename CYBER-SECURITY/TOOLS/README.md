#  Cyber Security Internship – Tools & Resources

This repository contains a curated list of essential tools and platforms used during the Cyber Security Internship for learning, practice, and hands-on experience.

---

##  Reconnaissance & OSINT Tools
- **Maltego** – Link analysis & investigation  
- **Sherlock** – Find social media accounts  
- **theHarvester** – Email/domain data collection  
- **HaveIBeenPwned** – Check data breaches  
- **Google Dorks** – Advanced search techniques  

---

##  Web Security Testing Tools
- **Burp Suite (Community)** – Intercept & modify requests  
- **OWASP ZAP** – Automated vulnerability scanner  
- **Nikto** – Web server scanner  
- **SQLMap** – SQL Injection testing  

---

##  Network & Scanning Tools
- **Nmap** – Port scanning & service detection  
- **Wireshark** – Network packet analysis  
- **Netcat** – Network debugging tool  

---

##  Password & Authentication Tools
- **John the Ripper** – Password cracking  
- **Hashcat** – Advanced hash cracking  
- **Hydra** – Brute force login testing  

---

##  Social Engineering & Phishing Tools *(Educational Use Only)*
- **SET (Social Engineering Toolkit)** – Phishing simulations  
- **GoPhish** – Phishing campaign testing  
- **Email Header Analyzer** – Check spoofed emails  

---

##  Vulnerability Practice Platforms
- **TryHackMe** – Beginner to advanced labs  
- **Hack The Box** – Real-world hacking labs  
- **PortSwigger Web Security Academy** – Web attack practice  

---

##  Development & Security Tools
- **VS Code** – Coding environment  
- **GitHub** – Code hosting & version control  
- **Python / JavaScript** – Tool building  

---

##  Dark Web & Threat Intelligence *(Awareness Only)*
- **Tor Browser** – Access onion sites (for study only)  
- **IntelX** – Data leak search  
- **Shodan** – Internet-connected device search  

---

##  Reporting & Documentation
- **Notion / Google Docs** – Report writing  
- **Canva** – Security awareness content  
- **OBS Studio** – Screen recording for demos  

---

##  Browser Extensions *(Very Useful)*
- **Wappalyzer** – Detect website technologies  
- **FoxyProxy** – Proxy management  
- **EditThisCookie** – Cookie testing  

---

##  Important Notes
- Use tools only in **legal lab/test environments**  
- Never scan or attack real systems **without permission**  
- Always follow **ethical hacking guidelines**  

---

##  Purpose of This Repository
- Learn practical cyber security tools  
- Perform hands-on labs  
- Build real-world security skills  
- Document internship progress
